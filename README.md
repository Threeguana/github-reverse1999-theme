# GitHub × Reverse 1999 Theme ✨
A custom userscript that transforms the standard GitHub interface into a vintage Art Deco masterpiece inspired by the time-traveling aesthetic of the game *Reverse 1999*.

This theme overhauls GitHub's Primer CSS with a magical atmosphere, featuring a carefully curated color palette and ethereal animations.

## Features
* **Vintage Art Deco Palette:** Replaces GitHub's default colors with Deep Navy, Antique Gold, Aged Cream, and Dusty Rose.
* **Elegant Typography:** Utilizes classic serif fonts (`Playfair Display`, `IM Fell English`, and `Crimson Pro`) for headings and text, mimicking old newspaper prints and magical archives.
* **Ethereal Animations:**
  * **Fade-In-Up:** Page content and cards gently float into view upon loading.
  * **Anti-Gravity Float:** The GitHub logo and user avatars slowly levitate, simulating the "Storm" anomaly.
  * **Pulse Glow:** Buttons emit a soft, magical golden heartbeat when hovered.
* **Refined Code Blocks:** Custom syntax highlighting that matches the vintage aesthetic without sacrificing readability.

## Installation
To use this theme, you need a userscript manager extension installed in your browser.

### Step 1: Install a Userscript Manager
Choose one of the following based on your browser:
* **Chrome / Edge / Brave:** [Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) or [Violentmonkey](https://chrome.google.com/webstore/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag)
* **Firefox:** [Tampermonkey](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/) or [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/)

### Step 2: Install the Theme
Once your extension is ready, simply click the link below to install the script automatically:

**[Click Here to Install the Theme](https://raw.githubusercontent.com/UsernameKamu/NamaRepoKamu/main/reverse1999-theme.user.js)**

*(Note: The userscript manager will open a new tab asking for your confirmation. Click **Install** to apply the script.)*

## Customization
If you want to tweak the colors or disable specific animations, you can edit the script directly from your Tampermonkey dashboard. The CSS variables are located at the very top of the stylesheet inside the `:root, [data-color-mode]` block.
